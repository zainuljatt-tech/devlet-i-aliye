## Netlify Drop (recommended)

1. Open **https://app.netlify.com/drop** in Chrome or Edge.
2. Sign up or log in (free — Google account works).
3. Drag the whole folder **`devlet-i-aliye`** onto the page.
   - Must include: all `.html` files, `style.css`, `main.js`, and the **`images`** folder.
4. Netlify gives you a link like `https://random-name.netlify.app`.
5. Optional: **Site settings → Domain management → Options → Edit site name** to get a nicer URL (e.g. `dao-genc-istanbul.netlify.app`).
6. Share that link on WhatsApp — everyone sees the full site with images.

## After you change the site

Drag the folder onto Netlify Drop again, or connect GitHub later for automatic updates.

## GitHub Pages (optional, later)

1. Install [Git for Windows](https://git-scm.com/download/win).
2. Create a repo on [github.com](https://github.com).
3. Upload the project and enable **Settings → Pages → Deploy from branch → main**.

---

**Note:** Do not upload only `index.html`. The live site needs the `images` folder and other pages too.
